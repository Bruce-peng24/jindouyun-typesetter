# jindouyun-typesetter

此项目的目标是：

实现一个包含以下功能的排版工具：

界面设计（三步操作）
1. [粘贴HTML区域] - 大文本框，支持粘贴HTML内容
2. [排版方案选择] - 预设4种常用的美观的排版模板
3. [导出按钮] - 一键生成docx文档

预设排版方案
简洁通用：日常办公文档
学术论文：适合论文、报告格式
商务文档：企业报告、方案文档
技术文档：API文档、技术手册

具体改造计划
第一阶段（当前阶段）：简化现有功能
移除复杂配置界面：删除所有高级排版设置
保留核心转换引擎：Pandoc转换能力
重构UI：极简三步骤界面


第二阶段：增强AI友好性
HTML预处理：自动清理AI生成的杂乱HTML标签
智能标签识别：更好处理标题、列表、代码块等
提供标准提示词：帮助用户获得更规范的HTML输出

第三阶段：优化用户体验
实时预览：转换前可预览效果
模板预览图：直观展示不同排版效果
批量处理：支持多个HTML片段同时转换
